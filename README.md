# vroomvroom.pcb
This is a top class, 2-key macropad because 1 F1-themed macropad was not enough so I made another one. This one is a lot less cool with only 2 mechanical keys, no display screen, but you get cool LEDs :)!!

The PCB is paired with a very long description of the RB19 and many sparks and F1 symbols because why not!!! You can use this to improve your workflow by 15% since there 2 EXTRA KEYSSS.

I was inspired to make this because I love F1 and Fallout merch okay bye!!
